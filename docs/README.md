# 구현 기능 목록

### 로또 관련 기능
* 로또 구입 기능
  * 단일 구매 기능
  * 유저가 소유한 금액에 따른 구매 기능(여러개 구매)
* 로또 추첨 기능
  * 추첨 번호 입력 기능
* 당첨 확인 기능
  * 유저가 구입한 로또 당첨 여부 확인
* 당첨금 및 수익률 계산 기능
  * 당첨 등수 별 통계 기능
  * 구입 금액과 당첨금에 따른 수익률 계산 기능

### 번호 생성 기능
* 로또 번호 생성 기능
  * 번호 생성 예외 처리

### 돈 관련 기능
* 돈 입금 기능
  * 유저 구입 금액 입력 기능
  * 구입 금액 입력 예외 처리
